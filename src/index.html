Reiterating some important concepts about Assignment2

Hello everyone,

As you plow on assignment2, here are some things to keep in mind as also discussed earlier in lecture:

  1. You cannot use 3rd party CSS framework.

  2. You cannot use 3rd party JavaScript framework such as JQuery etc.

  • However, you can use your own JavaScript functions written by you and which does not reference any 3rd party JavaScript frameworks.

  1. The due date for assignment2 is 12th March @ 11:59 pm.

  2. You can use the content (such as pdf, instructor name, TA name, etc. etc.) from either Anna’s website or our current website.

  3. Each user story can be 2-3 sentences. Concentrate on how the user will interact with your website. Also to think about here is the kinds of user (instructor or admin or student). Your user stories and mock diagram must be well considered and thorough. In your mock diagram, please do not use a screenshot of the actual web page. This is incorrect. The mock diagram happens first. At this point, you have a rough idea of how the webpage should work and how the user will interact with it.

  • At no point in the mocks should we see elements that reflect the actual webpage. Again use the wireframe mockup (referenced in the handout) to draw these out.
  • However, when you implement (when you start using HTML and CSS) these user stories on your web page, it is OK if you just concentrate from the student point of view.

  1. Here are some example of user stories, that I encourage you to read and get familiar with: https://www.mountaingoatsoftware.com/agile/user-stories

  2. Make sure to test and run your webpage on Google Chrome. If your webpage depends on some images and other resources, make sure to have them placed appropriately in the directory that you submit to us. Please see the assignment handout for this.

  3. I hope you can take advantage of all the office hours this week towards your assignment2 and for prep towards quiz2.

thanks, abbas
